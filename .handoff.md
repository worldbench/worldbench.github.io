## Paper-page content alignment

The page mirrors the latest paper. **Important rule**: when the paper's prose drops a specific number (e.g., "37% ceiling", "5.38 → 3.41"), the page should follow with qualitative language. We did a pass to remove most metrics from the stage card descriptions and use "the paper's framing" instead. Keep numbers in the appendix table evaluation column (those came from the LaTeX appendix tables verbatim).

If the user asks to "reformulate cards from the paper":
1. Pull from Overleaf (`cd /Users/lingdong/Documents/awesome-ai-auto-research-overleaf && git pull origin master`)
2. Re-read `sections/1_intro.tex`, `sections/2_creation.tex`, etc.
3. Update phase subtitles, motivation paragraphs, and stage descriptions to match
4. Tables in the Paper Collection section come from `tables/appendix_s{1-8}.tex` — the user wants these kept faithful to those LaTeX tables

- **Likes**: Apple/Anthropic-style minimalism, generous whitespace, editorial Fraunces serif, warm amber accent, expandable accordions for dense content
